# TamuAI — Telegram Bot 💛

Personal AI assistant built by Temesgen for Hawi.
Powered by Gemini AI · Hosted on Koyeb · Free forever.

## Setup

1. Clone and install
   npm install

2. Copy .env.example to .env and fill in your keys
   cp .env.example .env

3. Run locally
   npm start

## Commands

/start  — Start fresh conversation
/reset  — Clear conversation memory
/help   — Show help

## Deploy to Koyeb

Push to GitHub, connect repo on koyeb.com, set env variables, deploy.
